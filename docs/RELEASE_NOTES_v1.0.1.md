# StegoTool Pro v1.0.1 Release Notes

Release date: 2026-04-19

## Summary

StegoTool Pro v1.0.1 is a targeted stability release for video steganography output correctness.

## Fixes

- Fixed video encoding artifacts
- Fixed incorrect duration/timestamps
- Preserved audio in encoded video

## Notes

- Encode/decode workflows remain unchanged.
- Payload format and cryptographic behavior remain unchanged.
- This remains a local desktop Windows release.
